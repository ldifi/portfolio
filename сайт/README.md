# Сайт-портфолио (статическая страница)

## Запуск

Откройте `index.html` двойным кликом в браузере.

## Настройка контактов

В `index.html` замените:

- `mailto:yourmail@gmail.com` на ваш email
- `https://t.me/your_username` на ваш Telegram

## Картинки работ

Сейчас в карточках стоят стилизованные плейсхолдеры (в исходных данных есть только `imageHash` без самих файлов).
Если у вас есть изображения, скажите/скиньте названия файлов — подключу их как фон карточек.
